# Markdown Annotator

VS Code Markdown 标注插件 —— 添加高亮、注释，导出笔记。

## ✨ 功能

| 功能 | 说明 |
|------|------|
| ⚡ 快速高亮 | 一键高亮，不弹窗 |
| 🎨 选择颜色 | 选择高亮颜色（5种） |
| 💬 添加注释 | 为选中文字添加批注 |
| 📋 查看标注 | 列出所有标注，点击跳转 |
| 🗑️ 清除高亮 | 一键清除当前文件高亮 |
| 📤 导出标注 | 导出为 JSON 文件 |

---

## 📦 安装

去 [Releases](https://github.com/Jack-Long-2022/vscode-markdown-annotator/releases) 下载 `.vsix` 文件，然后：

1. VS Code 按 `Ctrl+Shift+P`
2. 输入 `Extensions: Install from VSIX...`
3. 选择下载的文件

## 🚀 三种激活方式

### 方式一：右键菜单（推荐）

```
┌─────────────────────────────────────┐
│  选中文字 → 右键 → 📝 Markdown 标注  │
│                                     │
│  ┌──────────────────────────────┐   │
│  │ ⚡ 快速高亮（默认颜色）       │   │
│  │ 🎨 添加高亮（选择颜色）       │   │
│  │ 💬 添加注释                  │   │
│  │ ────────────────────────     │   │
│  │ 📋 查看所有标注              │   │
│  │ 🗑️ 清除当前文件高亮          │   │
│  │ 📤 导出所有标注              │   │
│  └──────────────────────────────┘   │
└─────────────────────────────────────┘
```

### 方式二：命令面板

```
1. 按 Ctrl+Shift+P（Mac: Cmd+Shift+P）
2. 输入 "Markdown Annotator"
3. 选择要执行的命令

示例：
  - 输入 "快速高亮" → 执行快速高亮
  - 输入 "添加注释" → 执行添加注释
  - 输入 "标注" → 显示所有标注相关命令
```

### 方式三：快捷键（Alt 系列）

| 功能 | 快捷键 | 难度 |
|------|--------|------|
| ⚡ 快速高亮 | `Alt+H` | ⭐ 简单 |
| 🎨 选择颜色 | `Alt+Shift+H` | ⭐⭐ 中等 |
| 💬 添加注释 | `Alt+Shift+C` | ⭐⭐ 中等 |
| 📋 查看标注 | `Alt+L` | ⭐ 简单 |
| 🗑️ 清除高亮 | `Alt+Shift+D` | ⭐⭐ 中等 |
| 📤 导出标注 | `Alt+Shift+E` | ⭐⭐ 中等 |

> 💡 **为什么用 Alt？** Alt+字母 在 VS Code 中很少被占用，冲突概率最低。

## ⚙️ 配置

打开 VS Code 设置（`Ctrl+,`），搜索 `Markdown Annotator`：

| 设置项 | 默认值 | 说明 |
|--------|--------|------|
| `highlightColor` | `yellow` | 快速高亮默认颜色 |
| `alwaysShowColorPicker` | `true` | 关闭后快速高亮不弹窗 |
| `showInPreview` | `true` | 预览模式显示（实验性） |

## 📝 详细说明

### 高亮功能

- **快速高亮**：不弹窗，直接用默认颜色（可在设置中修改）
- **选择颜色**：弹出颜色选择器，5种颜色可选
  - 🟡 黄色（默认）
  - 🟢 绿色
  - 🔵 蓝色
  - 🩷 粉色
  - 🟠 橙色

> ⚠️ 高亮是临时装饰，关闭文件后会消失。如需保存，请用"导出标注"。

### 注释功能

注释会以 HTML 注释形式插入到文件中：

```markdown
这是一段被选中的文字
<!-- 💬 [2026-03-29 09:30] 这是我的注释 -->
```

- 注释在预览中不可见
- 注释会保存在文件中
- 可以用 `Ctrl+F` 搜索 `<!-- 💬` 找到所有注释

## ❓ 常见问题

### Q: 右键菜单没有 Markdown 标注？
A: 确保打开的是 `.md` 文件，插件只在 Markdown 中激活。

### Q: 快捷键不生效？
A: 可能与其他软件冲突，改用右键菜单或命令面板。

### Q: 高亮消失了？
A: 高亮是临时装饰，用"导出标注"保存记录。
