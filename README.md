# Markdown Annotator

VS Code Markdown 标注插件 —— 添加高亮、注释，导出笔记。

## ✨ 功能特点

| 功能 | 说明 |
|------|------|
| ⚡ 快速高亮 | `Ctrl+H` 一键高亮，不弹窗 |
| 🎨 选择颜色 | `Ctrl+Shift+H` 选择高亮颜色 |
| 💬 添加注释 | 为选中文字添加批注 |
| 📋 查看标注 | 列出所有标注，点击跳转 |
| 🗑️ 清除高亮 | 一键清除当前文件高亮 |
| 📤 导出标注 | 导出为 JSON 文件 |

---

## 📦 安装

### 方法一：直接安装 .vsix（推荐）

1. 去 [Releases](https://github.com/Jack-Long-2022/vscode-markdown-annotator/releases) 下载最新的 `.vsix` 文件
2. VS Code 按 `Ctrl+Shift+P` → 输入 `Extensions: Install from VSIX...`
3. 选择下载的文件，4. 重启 VS Code

## ⚙️ 快速高亮 vs 选择颜色

| 模式 | 快捷键 | 说明 |
|------|-------|------|
| 快速高亮 | `Ctrl+H` | 不弹窗，直接用默认颜色 |
| 选择颜色 | `Ctrl+Shift+H` | 弹出颜色选择器 |

> 💡 在设置中修改默认颜色：`markdownAnnotator.highlightColor`

## 🔧 快捷键冲突解决

### 方法一：自定义快捷键

```
1. Ctrl+K Ctrl+S（打开键盘快捷方式）
2. 搜索 markdownAnnotator
3. 双击命令，按下新快捷键
4. 回车确认
```

### 方法二：禁用快捷键

在设置中关闭 `markdownAnnotator.enableShortcuts`，然后：
- 用命令面板 `Ctrl+Shift+P`
- 用右键菜单

---

## ⚙️ 配置选项

打开 VS Code 设置（`Ctrl+,`），搜索 `Markdown Annotator`：

| 设置项 | 默认值 | 说明 |
|--------|--------|------|
| `highlightColor` | `yellow` | 默认高亮颜色 |
| `alwaysShowColorPicker` | `true` | 关闭后快速高亮不弹窗 |
| `showInPreview` | `true` | 预览模式显示（实验性） |
| `enableShortcuts` | `true` | 关闭后用命令面板 |
