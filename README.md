# Markdown Annotator

VS Code Markdown 标注插件 —— 添加高亮、注释，导出笔记。

## ✨ 功能

| 功能 | 说明 |
|------|------|
| ⚡ 快速高亮 | 一键高亮，直接使用默认颜色 |
| 🎨 选择颜色高亮 | 弹出颜色选择器，5种颜色可选 |
| 💬 添加注释 | 为选中文字添加批注（HTML 注释形式） |
| 📋 查看所有标注 | 列出当前文件所有标注，点击跳转 |
| 🗑️ 清除高亮 | 清除当前文件所有高亮 |
| 📤 导出标注 | 导出所有标注为 JSON 文件 |

---

## 📦 安装

1. 去 [Releases](https://github.com/Jack-Long-2022/vscode-markdown-annotator/releases) 下载 `.vsix` 文件
2. VS Code 按 `Ctrl+Shift+P`
3. 输入 `Extensions: Install from VSIX...`
4. 选择下载的文件，重启 VS Code

## 💬 注释功能

添加的注释会以 HTML 注释形式插入到文件中：

```markdown
这是一段被选中的文字
<!-- 💬 [2026-03-29 09:30] 这是我的注释 -->
```

**特点**：
- ✅ 注释在预览中不可见
- ✅ 注释会保存在文件中
- ✅ 可以用 `Ctrl+F` 搜索 `<!-- 💬` 找到所有注释

---

## ⚙️ 配置选项

打开 VS Code 设置（`Ctrl+,`），搜索 `Markdown Annotator`：

| 设置项 | 默认值 | 说明 |
|--------|--------|------|
| `highlightColor` | `yellow` | 快速高亮的默认颜色 |
| `alwaysShowColorPicker` | `true` | 高亮时是否显示颜色选择器 |
| `showInPreview` | `true` | 预览中是否显示标注（实验性） |

## 📤 导出格式

导出的 JSON 文件格式：

```json
{
  "exportedAt": "2026-03-29 09:30",
  "totalFiles": 1,
  "files": {
    "/path/to/file.md": {
      "annotationCount": 3,
      "annotations": [
        {
          "type": "highlight",
          "color": "yellow",
          "text": "高亮的文字",
          "timestamp": 1743212400000
        },
        {
          "type": "comment",
          "comment": "这是注释内容",
          "text": "被注释的文字",
          "timestamp": 1743212500000
        }
      ]
    }
  }
}
```

## ❓ 常见问题

### Q: 右键菜单没有标注选项？
A: 
1. 确保打开的是 `.md` 文件
2. ⚡ 快速高亮、🎨 选择颜色、💬 添加注释 —— **需要先选中文字**
3. 如果还是不行，尝试重启 VS Code

### Q: 高亮消失了？
A: 高亮是临时装饰效果，关闭文件后会消失。如需保存，请用"📤 导出所有标注"。

### Q: 注释在哪里？
A: 注释是 HTML 注释 `<!-- ... -->`，在编辑模式可以看到，预览模式不可见。

### Q: 想用快捷键？
A: 可以自己设置：
1. `Ctrl+K Ctrl+S` 打开键盘快捷方式
2. 搜索 `markdownAnnotator`
3. 双击命令，按下你想用的快捷键
